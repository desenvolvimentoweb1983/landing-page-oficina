# 🚗 Landing Page para Oficina Mecânica

Template de **Landing Page profissional para oficinas mecânicas**, desenvolvido com **HTML, CSS e JavaScript puro**.

O objetivo deste projeto é criar um **modelo reutilizável** que pode ser adaptado rapidamente para diferentes oficinas mecânicas.

Ideal para desenvolvedores que desejam **criar sites rápidos para negócios locais**.

---

# 🌐 Demonstração

Este projeto pode ser publicado facilmente usando **GitHub Pages**.

# ⚙️ Como usar este template

1. Faça download ou clone o repositório

```
git clone https://github.com/desenvolvimentoweb1983/landing-page-oficina.git
```

2. Abra o projeto no editor de código

Recomendado: Visual Studio Code

3. Edite o arquivo **script.js**

Altere as informações do cliente:

* nome da oficina
* telefone
* endereço
* serviços
* veículos atendidos

4. Substitua as imagens da pasta **img**

---

# 📱 Contato via WhatsApp

O botão de WhatsApp usa a seguinte estrutura:

```
https://wa.me/5511999999999
```

Basta alterar o número no arquivo **script.js**.

---

# 📍 Localização

O mapa utiliza **Google Maps Embed**.

Você pode alterar a localização substituindo o link dentro de:

```
mapa:"https://www.google.com/maps?q=LATITUDE,LONGITUDE&output=embed"
```
